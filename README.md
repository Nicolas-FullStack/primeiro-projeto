# 🚀 Primeiro Projeto Angular

Este foi meu primeiro contato com o framework Angular, onde explorei os conceitos básicos para construção de aplicações web modernas.

## 🌐 Projeto Online
👉 https://primeiro-angular-nikolas.web.app/hello-world

## 📌 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de aprender os fundamentos do Angular, incluindo:

- Estrutura de um projeto Angular
- Criação de componentes
- Data binding
- Uso de templates
- Organização de código

## 🛠️ Tecnologias utilizadas
- Angular
- TypeScript
- HTML
- CSS
- Firebase Hosting

## 🎯 Objetivo
Praticar e entender os conceitos iniciais do Angular, servindo como base para projetos mais avançados no futuro.
